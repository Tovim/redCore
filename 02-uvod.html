1. Instalace

1.1 Instalace z distribučního souboru

První naše kroky v hájemnství jazyka Red nezbytně začínají na záložce Downloads jeho domovské stránky red-lang.org.

Zde jsou k disposici binární spustitelné soubory pro systémy Windows, GNU/Linux a Mac OS-X z vývojových větví "last stable" (doporučeno) a novější "master". K disposici jsou rovněž komprimované verze ~.zip a ~.tar.gz zdrojových nebinárních souborů.

Instalace jazykového prostředí Red je dětinsky prostá. Binární soubor (red.exe) umístíme do libovolné složky počítače.

Při prvním poklepu soubor red.exe proběhne instalace kompilátoru, interpreta a potažmo otevření grafické konzoly Red Console. Při každém dalším poklepu na soubor red.exe se otevře přímo tato konzola.

Alternativně je možné pro evokaci interaktivního prostředí REPL (Read-Evaluate-Print-Load) použít systémovou konzolu (případně aplikaci Console 2.00):

> red --cli

--== Red 0.6.3 ==-- 
Type HELP for starting information. 
>>

Abychom měli přístup k souboru red.exe z kteréhokoli diskového oddílu či složky, vložíme si cestu k němu do systémové proměnné PATH. Musíme však instalační soubor přejmenovat na red-exe.exe a připojit k němu nový soubor red.bat s tímto obsahem:

"%~dp0red-exe.exe" %*

V konzole pak můžeme odkudkoli vyvolat okno "Red Console", například: 

F:\> red
nebo CLI verzi příkazem: 
F:\> red --cli

1.2. Instalace ze zdrojových kódů

Poklep na červenou stužku v levém horním rohu domovské stránky nás přenese na stránky github.com/red/red, kde je k disposici aktuální verze zdrojového kódu, kterou lze stáhnout nebo klonovat do repozitáře Git v počítači.

Postup instalace je popsán v textu Readme na výše uvedené stránce. Při této instalaci je potřebný překladač Rebolu, na jehož zdroj jsou pro Windows, Linux, Mac OS X, FreeBSD, OpenBSD a Solaris v uvedeném textu odkazy.

Tato informace je zde uvedena jenom pro úplnost, neboť pro začínajícího adepta programování v jazyce Red nemá velký význam. Tato alternativa je určena pro přispívající vývojáře.

2. Interaktivní konzola

Red patří k těm přívětivým jazykům, které svým uživatelům poskytují interaktivní prostředí, buď v samostatné grafické konzole (ve Windows) nebo v systémové konzole příkazem red --cli.

Při práci v interaktivním prostředí Redu je vhodné vědět o existenci některých příkazů:

  • Prostředí opustíme příkazem quit nebo pouhým q.
  • Nápovědu vyvoláme příkazem help <cosi> nebo pouhým ? <cosi>.

Dobrým zdrojem heslovitých informací je také internetová stránka Red by Example.

3. Obligátní Hello světe!

Red vytvoří zobrazení tohoto pozdravu dětinsky prostým způsobem v příkazovém řádku:

>> print "Hello World!"
Hello World!

Totéž umí v grafickém provedení:

>> view [text "Hello World!"]

Vestavěný grafický systém Red/GUI vytvoří okno s nápisem "Hello World!":

Podobně jednoduchým způsobem umí vytvořit i interaktivní okno s polem pro zadání vlastního jména: 

>> view [name: field button "Hi" [print ["Hi" name/text]]]

4. Kompilace skriptu

Totéž, co jsme napsali do příkazového řádku konzoly, můžeme zapsat do textového souboru s koncovkou ~.red.
Zdrojové soubory mají povinné záhlaví Red []. V interaktivním prostředí je toto záhlaví implicitně přítomno. Text před tímto záhlavím je ignorován, text za záhlavím tvoří vlastní zdrojový kód.
Uvnitř hranatých závorek se nachází průvodní informace skriptu a provozní pokyny.

V textovém editoru vytvoříme soubor s názvem hello.red s tímto obsahem:

Red [Title: "Jednoduchý skript pro Hello World"]
    
print "Hello World!"

Ve složce se souborem hello.red uplatníme v systémové konzole tento příkaz: 

> red -c hello.red
;; otevře se kompilátor:
=== Red Compiler 0.6.1 ===
Compiling F:\Codetest\Red\hello.red ...
Compiling to native code...
... output file : F:\Codetest\Red\hello.exe
;; provoz se vrátí do původní konzoly:
 F:\Codetest\Red>

Vytvořený binární soubor hello.exe spustíme z příkazového řádku:

> hello
Hello World!

Spustíme-li tento soubor poklepem na název v Průzkumníku, otevře se systémová konzola ale okamžitě se zavře, s čímž si však poradíme ...

Popsaný způsob platí pro kompilaci v lokálním systému Windows (x86). Red aktuálně podporuje 11 dalších platforem.

Přechod na jinou platformu se uskuteční následným způsobem:

$ red -c -t Windows hello.red
> red -c -t Darwin hello.red
> red -c -t Linux-ARM hello.red

Zdrojový kód pro grafickou aplikaci vyžaduje avízo Needs: 'View v záhlaví: 

Red [Needs: 'View]
    
view [name: field button "Hi" [print ["Hi" name/text]]]

Uložený soubor hello-gui.red kompilujeme stejným způsobem jako předchozí soubor hello.red. Binární soubor spustíme z příkazového řádku nebo poklepem názvu. V druhém případě se nejprve otevře konzola a poté se vytvoří grafický prvek na obrazovce.
