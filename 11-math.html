1. Přehled

Red poskytuje všeobecný výběr matematických a trigonometrických operací. Mnohé operátory pracují s vícerými datovými typy, včetně typů integer, float, money, tuple, time a date. Some of these datatypes may even be mixed, or coerced.

3. Pořadí výpočtu

Pro vyhodnocení matematických výrazů platí dvě pravidla:

  • Výrazy se vyhodnocují zleva do prava.
  • Operátory mají přednost před funkcemi.

Vyhodnocení výrazu zleva doprava probíhá nezávisle na typu operátoru, například:

print 1 + 2 * 3
9   ; nikoliv 7 !

Important Note

The way mathematical expressions are evaluated from left to right regardless of the operator is different than many other computer languages. Many languages have rules of precedence that you must remember that determine the order of evaluation of operators. For example, a multiply is done before an add. Some languages have 10 or more such rules.

In RED, rather than requiring users to remember the precedence of operators, you only need to remember the left-to-right rule. More importantly, for advanced code such as expressions that handle expressions (in reflection, for example) you do not need to reorder terms based on precedence. The evaluation order is kept simple.

For most math expressions, the left-to-right evaluation rule works quite well and is easy to remember. However, because this rule is different from other languages, it can be a source of programming errors, so watch out.

The best solution is to check your work. You can also use parentheses if necessary to clarify your expression (see below), and you can always type your expression at the console to verify your result.

Potřebujeme-li aby výpočet probíhal v jiném pořadí, použijeme zápis s jiným pořadím nebo použijeme závorky:

print 2 * 3 + 1
7
print 1 + (2 * 3)
7

Jsou-li ve výrazu funkce i operátory, vyhodnocují se nejprve operátory, potom funkce:

print absolute -10 + 5
5

V uvedené ukázce je nejprve proveden součet a jeho výsledek je zadán funkci absolute.

V další ukázce:

print 10 + sine 30 + 60
11.0

je výraz vyhodnocen v tomto pořadí jakoby probíhal zprava do leva:

30 + 60 => 90       
sine 90 => 1
10 + 1 => 11
print

Pro přednostní vyhodnocení výrazu sine 30 použijeme závorky:

print 10 + (sine 30) + 60
70.5

nebo přeskupíme výraz:

print 10 + 60 + sine 30
70.5

6. Relační funkce

Všechne relační funkce vracejí buď true nebo false.

6.1 equal

Výrazy vracejí true, jsou-li obě hodnoty stejné.

value1 = value2

equal? value1 value2

Pracuje s typy integer, float, (money), time, (date), tuple, char a series.

print now/time = 15:30
false
print equal? 111.112.111.111  111.112.111.111
true
print #"B" = #"B"
true
print equal? "a b c d" "A B C D"
true

6.2 greater

Výrazy vracejí true, je-li první hodnota větší než druhá.

value1 > value2

greater? value1 value2

Pracuje s typy integer, float, (money), time, (date), tuple, char a series.

print now/time > 15:30
true (or false)
print greater? 113.111.111.111   111.112.111.111
true
print #"C" > #"B"
true
print greater? [12 23 34] [12 23 33]
true

6.3 greater-or-equal

Výraz vrací true, je-li první hodnota větší nebo rovna druhé.

value1 >= value2

greater-or-equal? value1 value2

Pracuje s typy integer, decimal, (money), time, (date), tuple, char and series.

print now/time >= 12:00
true
print greater-or-equal? 111.112.111.111   111.111.111.111
true
print #"B" >= #"A"
true
print greater-or-equal? [b c d e] [a b c d]
true

6.4 lesser

Výraz vrací true, je-li první hodnota menší než druhá.

Pracuje s typy integer, float, (money), time, (date), tuple, char a series:

6.5 lesser-or-equal

Výraz vrací true, je-li první hodnota menší nebo stejná jako druhá.

Výraz vrací true, nejsou-li hodoty stejné.

Pracuje s typy integer, float, (money), time, (date), tuple, char a series.

6.7 same

Výraz vrací true, odkazují-li dvě slova na stejnou hodnotu. Vhodné pro zjištění, zda dvě slova odkazují na stejný index v hodnotě typu series.

value1 =? value2

same? value1 value2

Pracuje se všemi datovými typy.

reference-one: "abcdef"
reference-two: reference-one
same? reference-one reference-two
true
reference-one: next reference-one
same? reference-one reference-two
false
reference-two: next reference-two
same? reference-one reference-two
true
reference-two: copy reference-one
same? reference-one reference-two
false

6.8 strict-equal

Výraz vrací true, jsou-li obě hodnoty striktně stejné. Lze použít jako "case-sensitivní" verzi operátoru equal? pro řetězce a k rozlišení mezi čísly typu integer a float, mají-li stejnou hodnotu.

value1 == value2

strict-equal? value1 value2

Pracuje se všemi datovými typy.

print strict-equal? "abc" "ABC"
false
print equal? "abc" "ABC"
true
print strict-equal? "abc" "abc"
true
print strict-equal? 1 1.0
false
print equal? 1 1.0
true
print strict-equal? 1.0 1.0
true

9. Logické funkce

Logické funkce lze použít pro logické hodnoty a pro některé skalární hodnoty včetně hodnot typů integer, char, tuple a bitset. Při práci s logickými hodnotami vrací logická funkce booleovu hodnotu.

QUEER: When working with other types of values, the logic functions on the bits.

9.1 and

Funkce and porovná dvě logické hodnoty a vrací true, jsou-li obě pravdivé:

print (1 < 2) and (2 < 3)
true
print (1 < 2) and (4 < 3)
false

Při použití s celými čísly, porovnává funkce and bitové vyjádření obou čísel a vrací výsledek jako celé číslo:

print 3 and 5             011 and 101 -> 001 -> 1
1

9.2 or

Funkce or porovná dvě logické hodnoty a vrací true, je-li alespoň jedna pravdivá, nebo false, jsou-li obě nepravdivé. false :

print (1 < 2) or (2 < 3)
true
print (1 < 2) or (4 < 3)
true
print (3 < 2) or (4 < 3)
false

Při použití s celými čísly, porovnává funkce or bitové vyjádření obou čísel a vrací výsledek jako celé číslo:

print 3 or 5       011 or 101 -> 111 -> 7
7

9.3 xor

Funkce xor porovná dvě logické hodnoty a vrací true tehdy a jen tehdy, mají-li oba výrazy různou pravdivostní hodnotu.

print (1 < 2) xor (2 < 3)
false
print (1 < 2) xor (4 < 3)
true
print (3 < 2) xor (4 < 3)
false

Při použití s celými čísly, porovnává funkce xor bitové vyjádření obou čísel a vrací výsledek jako celé číslo:

print 3 xor 5            011 xor 101 -> 110 -> 6
6

9.4 complement

Funkce complement vrací logický nebo v bitech vyjádřený doplněk zadané hodnoty. It is used for bitmask integer numbers and inverting bitsets.

print complement true
false
print complement 3            011 -> 100 -> -4
-4

9.5 not

Funkce not vrací opak zadané pravdivostní hodnoty. Neprovádí číselné bitwise operace. 

print not true
false
print not (2 < 1)
true

10. Errors

Matematické chyby jsou hlášeny při výskytu ilegální operace nebo při přetečení či podtečení. Lze se setkat s těmito chybami:

10.1 Pokus o dělení nulou

1 / 0
** Math Error: Attempt to divide by zero
** Where: connect-to-link
** Near: 1 / 0

10.2 Číselné přetečení

Použití příliš velkého čísla signalizuje Red odezvou:

1E+700 + 1E+400
== 1.#INF

10.3 Požadované kladné číslo

Pokus o zpracování záporného čísla operátorem, který přijímá pouze kladná čísla: 

log-10 -1
== 1.#NaN

10.4 Nekompatibilní operátor s typem hodnoty

Inkriminovaný bývá druhý operand:

10:30 + 1.2.3
** Script Error: time! type is not allowed here
